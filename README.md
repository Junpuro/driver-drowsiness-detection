# AI Driver Drowsiness Detection

## Description
This project detects driver drowsiness in real-time using computer vision.

## Technologies
- Python
- OpenCV
- MediaPipe

## Features
- Eye detection using EAR
- Real-time webcam processing
- Audio alert when drowsy

## Run
pip install -r requirements.txt  
python main.py
